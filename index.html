gutenberg: gut is good!

Gutenberg is a massive template printing, aka scaffolding, tool for Erlang. Like rails generate or yeoman but for Erlang. Gutenberg was highly insipred by lfeetool.

tl;dr. install

curl

curl -L -O https://github.com/unbalancedparentheses/gut/raw/master/bin/gut
chmod +x ./gut
sudo mv ./gut /usr/local/bin/

wget

wget https://github.com/unbalancedparentheses/gut/raw/master/bin/gut
chmod +x ./gut
sudo mv ./gut /usr/local/bin/
